# CLAUDE.md

## Project Overview

Landing page project ("עמוד נחיתה - סטורי על סטרואידים").

## Custom Skills

### UI Expert — High-End UI Designer
Your goal is to generate modern, premium, and visually stunning landing pages using Tailwind CSS.

#### Design Principles:
- **Layout:** Use Bento Grids, clean whitespace, and asymmetrical layouts where appropriate.
- **Visuals:** Implement Glassmorphism (backdrop-blur), soft shadows (shadow-xl), and subtle gradients (mesh gradients).
- **Typography:** Use modern sans-serif fonts (like Inter or Geist) with clear hierarchy and optimized line-height.
- **Components:** Design high-end buttons with hover states, card layouts with borders that glow on hover, and sticky navigation bars.

#### Technical Constraints:
- Always use Tailwind CSS arbitrary values for precision if needed (e.g., `bg-[#color]`).
- Use Lucide React for consistent, thin-stroke icons.
- Ensure 100% responsiveness (Mobile-first approach).
